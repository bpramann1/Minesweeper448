tile¶

class tile.Tile(i, j)¶

A tile button on the minesweeper board

Holds all data about the tile

Parameters:
  • i (int) – The row of the tile
  • j (int) – The column of the tile
flagMine()¶

Toggles the flag on the tile.

Returns:The number to increment the number of mines found, 1 if mine flagged, -1 if mine unflagged, 0 if not a mine
Return type:Int
flip()¶

Checks whether the tile has already been click and if not call update()

Returns:False if already clicked, True if not
Return type:bool
getCount()¶

Returns number of mines in surrounding 8 neighbor tiles

Returns:count of near by mines
Return type:int
getIndices()¶

Returns row and col location of the tile

Returns:row and col of the tile
Return type:(RETURN TYPE)
incCount()¶

Increments near by mine count by 1

Returns:always True
Return type:bool
isFlipped()¶

Returns if tile has been flipped

Returns:true if already flipped, false otherwise
Return type:bool
isMine()¶

Returns if tile is a mine

Returns:true if tile is a mine, false otherwise
Return type:bool

setMine()¶

Attempts to place mine on tile.

Returns:false if tile is already a mine, true otherwise

Updates the state of the button. Called if the tile was left clicked for the first time

Returns:None
